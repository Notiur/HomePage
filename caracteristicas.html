La Interfaz Principal

Una interfaz totalmente adaptable. Fue algo fundamental que se tuvo en cuenta al realizar el diseño básico de la aplicación web. Notiur está basado en Bootstrap 3 un Framework HTML 5 y CSS 3 para crear un estilo responsivo. De esta forma no importa desde que dispositivo se intente acceder a la aplicación web de Notiur. Siempre se tendrá la mejor experiencia posible.
